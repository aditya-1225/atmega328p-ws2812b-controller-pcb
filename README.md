This PCB uses an ATmega328P microcontroller to control an addressable WS2812B LED strip.

## Controls
The PCB includes three push buttons:
1. Increase LED brightness  
2. Decrease LED brightness  
3. Change the LED pattern

## Connections
- The LED strip connects to the PCB via a screw terminal (J1).
- The board is programmed using an 8-pin header connector (J2).
